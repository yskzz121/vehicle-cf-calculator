**バージョン**: 1.0
**最終更新日**: 2026-03-10
**開発元**: U&I 株式会社 コンサルティング事業部

## 1. 概要

### 1.1 目的
法人・個人事業主が社用車を取得する際の7つの方法を、キャッシュフロー・節税効果・売却収入を含めた「実質コスト」で比較分析するシミュレーションツール。

### 1.2 対象ユーザー
- U&I 株式会社 コンサルティングチーム
- 法人の財務・経理担当者
- 個人事業主

### 1.3 技術構成
| 項目 | 内容 |
|------|------|
| 形式 | 単一HTMLファイル（CSS/JS埋め込み） |
| 外部依存 | Google Fonts（Noto Sans JP / Lora / JetBrains Mono）、html2pdf.js（CDN） |
| 車両DB | `vehicles_data.json`（GitHub Pages配信、370車種） |
| ホスティング | GitHub Pages（`yskzz121.github.io/vehicle-cf-calculator`） |
| ファイル構成 | `vehicle_cf_calculator.html` = `index.html`（同一内容） |

### 3.2 Step 01: 車両選択

#### DBパネル
| 操作 | 内容 |
|------|------|
| メーカー選択 | 15メーカーからプルダウン |
| モデル選択 | メーカー内のモデルを動的表示 |
| グレード選択 | 価格・燃料タイプ付きで動的表示 |

グレード選択時に以下を自動入力:
- 車両本体価格、諸費用
- 任意保険料、燃料費（燃費・燃料タイプから算出）
- 売却想定額（残価率 × 取得総額）
- エコカー減税・補助金各項目

#### 新車/中古車トグル
| 状態 | 動作 |
|------|------|
| 新車（デフォルト） | 通常の耐用年数3年、DB価格・エコ減税適用 |
| 中古車 | 経過年数スライダー表示、エコ減税リセット、諸費用18万に変更、車検費用増加 |

#### 中古車特例（耐用年数計算）
```
経過年数 < 法定耐用年数(6年) の場合:
  耐用年数 = max(2, floor((6 - 経過年数) + (経過年数 × 0.2)))

経過年数 >= 6年 の場合:
  耐用年数 = max(2, floor(6 × 0.2)) = 2年
```

| 経過年数 | 耐用年数 | 定率法償却率 |
|----------|----------|------------|
| 1年 | 5年 | 0.400 |
| 2年 | 4年 | 0.500 |
| 3年 | 3年 | 0.667 |
| 4年 | 3年 | 0.667 |
| 5年 | 2年 | 1.000 |
| 6年以上 | 2年 | 1.000 |

#### 手動入力フォーム
| フィールド | デフォルト | 備考 |
|-----------|----------|------|
| 車両名称 | （空） | 自由入力 |
| 車両本体価格 | （空） | DB選択時自動入力 |
| 諸費用・登録費 | 290,000円 | 中古車時180,000円 |
| オプション | 0円 | |
| メンテ保証パック | 0円 | |
| 取得総額 | 自動計算 | 上記4項目の合計 |

### 3.3 Step 02: 共通パラメーター

| パラメーター | 入力形式 | 範囲 | デフォルト |
|-------------|---------|------|----------|
| 法人実効税率 | スライダー | 15〜40% | 25% |
| 個人事業主 所得税+住民税 実効税率 | スライダー | 15〜55% | 30% |
| 業務使用割合 | スライダー | 50〜100% | 80% |
| 比較期間 | スライダー | 3〜10年 | 5年 |
| 任意保険料 | 数値入力 | — | 200,000円/年 |
| 燃料費 | 数値入力 | — | 144,000円/年 |
| 車検・法定費用 | 数値入力 | — | 310,000円（期間合計） |
| 修繕積立 | 数値入力 | — | 250,000円（期間合計） |
| N年後売却想定額 | 数値入力 | — | 比較期間連動（自動計算） |

**比較期間変更時の連動**: 売却想定額がDBの `residual_rates` に基づき自動再計算される。

#### エコカー減税・補助金
| 項目 | 対象 | 上限目安 |
|------|------|---------|
| CEV補助金 | EV/PHEV/FCV | EV最大85万、PHEV最大55万 |
| エコカー減税・重量税軽減 | EV/HEV/クリーンディーゼル | 免税〜50%軽減 |
| 環境性能割 軽減額 | EV/PHEV | 非課税〜3% |
| グリーン化特例 | EV/PHEV | 翌年度約75%軽減 |
| 自治体独自補助金 | 自治体による | 東京都EV最大45万 等 |

**エコ補助金の分類**:
- `ecoSubsidy`（取得コスト軽減）= CEV + 環境性能割 + 自治体補助
- `ecoRunning`（維持費軽減）= 重量税軽減 + グリーン化特例（期間累計）

## 4. 取得方法（7メソッド）

### 4.1 法人 現金一括（cash）
| 項目 | 内容 |
|------|------|
| 色 | グリーン |
| 追加パラメーター | なし |
| 所有権 | あり |
| 売却 | あり |
| 損金算入 | 減価償却（業務按分）+ 維持費（業務按分） |

**計算式**:
```
維持費 = 保険 + 燃料 + 車検 + 修繕 − エコ維持費軽減
損金算入額 = 減価償却(業務按分) + 維持費×業務割合
節税効果 = 損金算入額 × 法人税率
総支出 = 取得総額 + 維持費 − 補助金
実質コスト = 総支出 − 節税効果 − 売却収入
```

### 4.2 法人 銀行借入（bank）
| 項目 | 内容 |
|------|------|
| 色 | アンバー |
| パラメーター | 年利（0.5〜8%、デフォルト2.5%）、返済回数（12〜84回、デフォルト60回）、頭金 |
| 所有権 | あり |
| 売却 | あり |
| 損金算入 | 減価償却 + 利息（業務按分）+ 維持費（業務按分） |

**計算式**:
```
借入額 = 取得総額 − 頭金
月額返済 = PMT(年利, 返済回数, 借入額)
支払総額 = 月額返済 × 返済回数 + 頭金
金利 = 支払総額 − 取得総額
損金算入額 = 減価償却(按分) + 金利×業務割合 + 維持費×業務割合
節税効果 = 損金算入額 × 法人税率
実質コスト = 取得総額 + 維持費 + 金利 − 補助金 − 節税効果 − 売却収入
```

### 4.3 個人 残価ローン（resi）
| 項目 | 内容 |
|------|------|
| 色 | パープル |
| パラメーター | 年利（0.5〜8%、デフォルト2.49%）、返済回数（12〜84回、デフォルト59回）、残価設定額 |
| 所有権 | なし（残価精算まで） |
| 売却 | なし（返却/残価精算） |
| 損金算入 | なし（個人名義、所得控除不可） |

**計算式**:
```
借入額 = 取得総額 − 残価設定額
月額返済 = PMT(年利, 返済回数, 借入額)
支払総額 = 月額返済 × 返済回数
金利 = 支払総額 − 借入額
維持費 = 保険 + 燃料 + 車検×0.5 + 修繕 − エコ維持費軽減  ※車検半額
実質コスト = 支払総額 + 維持費 − 補助金
```

### 4.4 法人 リース（lease）
| 項目 | 内容 |
|------|------|
| 色 | レッド |
| パラメーター | 想定金利（0.5〜8%、デフォルト3.0%）、リース回数（12〜84回、デフォルト60回）、メンテ加算 |
| 所有権 | なし |
| 売却 | なし |
| 損金算入 | リース料全額（業務按分）+ 維持費（業務按分） |

**計算式**:
```
月額リース料 = PMT(年利, 回数, 取得総額) + メンテ加算
リース料合計 = 月額 × 回数
金利 = リース料合計 − 取得総額
維持費 = 保険 + 燃料 + 車検×0.5 − エコ維持費軽減  ※車検半額、修繕なし
損金算入額 = リース料合計×業務割合 + 維持費×業務割合
節税効果 = 損金算入額 × 法人税率
実質コスト = リース料合計 + 維持費 − 補助金 − 節税効果
```

### 4.5 法人 ローン（loan）
| 項目 | 内容 |
|------|------|
| 色 | ブルー |
| パラメーター | 年利（0.5〜8%、デフォルト2.0%）、返済回数（12〜84回、デフォルト60回）、頭金 |
| 所有権 | あり |
| 売却 | あり |
| 損金算入 | 減価償却 + 利息（業務按分）+ 維持費（業務按分） |

**計算式**: 法人銀行借入と同一ロジック（金利デフォルトが異なる）

### 4.6 個人 現金一括（pcash）
| 項目 | 内容 |
|------|------|
| 色 | ティール |
| 追加パラメーター | なし |
| 所有権 | あり |
| 売却 | あり |
| 必要経費 | 減価償却（業務按分）+ 維持費（業務按分） |

**計算式**:
```
維持費 = 保険 + 燃料 + 車検 + 修繕 − エコ維持費軽減
必要経費額 = 減価償却(按分) + 維持費×業務割合
節税効果 = 必要経費額 × 個人事業主税率
実質コスト = 取得総額 + 維持費 − 補助金 − 節税効果 − 売却収入
```

### 4.7 個人 銀行ローン（pbank）
| 項目 | 内容 |
|------|------|
| 色 | オレンジ |
| パラメーター | 年利（0.5〜8%、デフォルト2.5%）、返済回数（12〜84回、デフォルト60回）、頭金 |
| 所有権 | あり |
| 売却 | あり |
| 必要経費 | 減価償却 + 利息（業務按分）+ 維持費（業務按分） |

**計算式**:
```
必要経費額 = 減価償却(按分) + 金利×業務割合 + 維持費×業務割合
節税効果 = 必要経費額 × 個人事業主税率
実質コスト = 取得総額 + 維持費 + 金利 − 補助金 − 節税効果 − 売却収入
```

## 5. 法人/個人の税制上の違い

| 項目 | 法人（cash/bank/lease/loan） | 個人事業主（pcash/pbank） | 個人（resi） |
|------|---------------------------|------------------------|------------|
| 経費区分 | 損金算入 | 必要経費 | なし |
| 適用税率 | 法人実効税率 | 所得税+住民税 実効税率 | — |
| 減価償却 | 全額損金 × 業務按分 | 全額必要経費 × 業務按分 | 不可 |
| 利息 | 損金算入 × 業務按分 | 必要経費 × 業務按分 | 不可 |
| リース料 | 全額損金 × 業務按分 | — | — |
| 維持費 | 損金算入 × 業務按分 | 必要経費 × 業務按分 | 不可 |

> **注意**: 個人の「必要経費」は個人事業主が対象。給与所得者は車購入による税メリットなし。

## 6. 主要計算ロジック

### 6.1 PMT関数（元利均等返済）
```javascript
function pmt(annualRate, nm, pv) {
  if (annualRate === 0) return pv / nm;
  const r = annualRate / 12;
  return pv * r * Math.pow(1+r, nm) / (Math.pow(1+r, nm) - 1);
}
```

### 6.2 減価償却（定率法・200%定率法）
```javascript
function calcDepr(total, biz, usefulLife, rate) {
  usefulLife = usefulLife || 3;  // 新車:3年
  rate = rate || 0.667;          // 新車:0.667
  let rem = total, s = 0;
  for (let i = 0; i < usefulLife; i++) {
    const d = rem * rate;
    s += d;
    rem -= d;
  }
  s += rem - 1;  // 備忘価額1円を残す
  return { deprSum: s, depr80: s * biz };
}
```

### 6.3 残価率（getResidualRate）
```javascript
function getResidualRate(car, years) {
  // residual_rates配列 [1年目, 2年目, ..., 10年目]
  if (car.residual_rates && car.residual_rates.length >= years) {
    return car.residual_rates[years - 1];
  }
  // フォールバック: residual_ratio_5yr から補間
  if (car.residual_ratio_5yr != null) {
    const r5 = car.residual_ratio_5yr;
    const annualDecay = Math.pow(r5, 1/5);
    return Math.max(0.05, Math.pow(annualDecay, years));
  }
  return null;
}
```

### 6.4 実質コスト計算フロー（ウォーターフォール）
```
＋ 車両取得額（or リース料合計）
＋ 金利・手数料
＋ 維持費（保険 + 燃料 + 車検 + 修繕）
━━━━━━━━━━━━━━━━━━━━━━
＝ 総支払額

− エコ維持費軽減
− 補助金（CEV等）
− 節税効果 / 必要経費控除
− 売却収入
━━━━━━━━━━━━━━━━━━━━━━
＝ 実質コスト（ネット）
```

## 7. 結果表示

### 7.1 結果バナー

- 実質コスト（最終行、最安はグリーンハイライト）
- 月額表示

### 7.3 タブ表示（4タブ）

#### タブ1: 詳細CF比較
全メソッドの横並び比較テーブル。各行でBest/Worstをカラーハイライト。

セクション構成:
1. 月次支払い（月額、支払合計、金利）
2. 維持費（保険、燃料、車検、修繕）
3. 損金算入・必要経費（減価償却、利息、リース料、維持費の業務按分、合計、節税効果）
4. エコカー減税・補助金
5. 売却収入
6. **実質コスト（最終行）**

#### タブ2: グラフ
- 実質コスト比較（横棒グラフ）
- 節税・必要経費控除効果比較（横棒グラフ）

#### タブ3: 非財務評価
7項目の定性評価テーブル（良/中/難の3段階タグ + 5段階星評価）:

| 評価項目 | 内容 |
|---------|------|
| 初期現金流出 | 一括払いの負担 |
| 月次固定支出 | 毎月の支払い負担 |
| 所有権・資産性 | 車両資産の帰属 |
| 節税・経費控除 | 税制メリットの大きさ |
| 途中売却の自由度 | 中途処分の柔軟性 |
| 融資枠への影響 | 借入余力への影響 |
| 修繕リスク | 修繕費負担 |

#### タブ4: 推奨コンパス
5つの観点から最適メソッドを推奨:
1. 法人 節税最大化
2. 個人事業主 経費控除最大
3. 総コスト最小
4. 現金温存
5. 月次負担最小

加えて「必須確認事項（税理士・顧問確認項目）」を掲載。

### 8.4 残価率の設定基準
ブランド・車種タイプ別に10年分の残価率を設定。以下はティア例:

| ティア | 代表車種 | 1年 | 3年 | 5年 | 7年 | 10年 |
|--------|---------|-----|-----|-----|-----|------|
| 最高 | Jimny | 88% | 65% | 50% | 40% | 34% |
| 高 | Land Cruiser, G-Class | 85% | 60% | 45% | 35% | 28% |
| 中 | 一般輸入車 | 76% | 53% | 38% | 27% | 15% |

## 12. 前提条件・免責事項

フッターに表示される免責事項:

- 減価償却は**定率法**（新車3年/率0.667、中古車は特例適用）を前提
- 維持費は入力値を比較期間で按分
- リース月額は元利均等計算（実際のリース会社見積もりと異なる場合あり）
- 個人メソッドは**個人事業主**を前提（給与所得者は税メリットなし）
- 本ツールの計算結果はあくまで参考値
- 最終的な税務・財務判断は担当税理士・顧問へ確認要
